# Unigram

A simple yet powerful graphing tool.  
Turn the universe into your own diagram.


## Features
- Easily added to any C# program
- Logarithmic support for both the X- and Y-Axis
- Dynamic camera movement
- Simple graph analysis
- Colorful graphs (with default windows colors xd)
- Draw your graphs into a button. Draw them into a button. Do it. Do it now. Very useful!
